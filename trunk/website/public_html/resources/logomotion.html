Game Summary

2011 FRC Short Game Description

PDF, 434 KB

Created by FIRST

Uploaded May 09 2011

A one-page summary of the Logomotion game created by FIRST. In essence, the objective of the game is to make the robot hang inflatable tubes on the opposite end of your field. During Autonomous mode, the robot has the opportunity to try to attempt to place an 'ubertube' on a peg.

Near the end of the match, the robot can attempt to maneuver next to a pole then deploy a mini-robot which will attempt to climb up the pole as quickly as possible.
